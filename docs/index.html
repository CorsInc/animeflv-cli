animeflv-cli

Busca anime y obtén links de streaming desde tu terminal.
Sin navegador, sin Puppeteer, solo Node.js.

$ git clone https://github.com/CorsInc/animeflv-cli.git
$ cd animeflv-cli && npm install
$ node index.js search "dragon ball"

¿Qué hace?

🔍

Buscar anime

Encuentra cualquier anime por nombre con búsqueda directa en AnimeFLV.

📋

Info + episodios

Muestra sinopsis, géneros, estado y lista completa de episodios.

🔗

Links de streaming

Obtén enlaces directos a múltiples servidores por episodio.

📦

Sin dependencias pesadas

Solo Axios + Cheerio. Nada de navegadores, Puppeteer ni Selenium.

⚡

Rápido

Scraping ligero desde el HTML. Respuestas en segundos.

🖥️

CLI nativa

Funciona en cualquier sistema con Node.js instalado.

Comandos

ComandoDescripción
search <query>Busca anime por nombre
info <slug>Muestra info del anime + lista de episodios
links <slug> <ep>Obtiene links de streaming de un episodio
all <slug>Info + links de todos los episodios
